# Changelog

All notable changes to pixpic will be documented in this file.

## [1.0.0] - 2025-05-25

### Features
- 7 种点样效果（标准、复杂、科技、自然、极简、几何、星辰）
- 多色板支持（原图选色、千禧梦境、开心五彩、自然色系、亚比荧光）
- 贴纸系统（边框、花束、植物、动物、Y2K、装饰）
- 贴纸编辑（点样/色板/透明度）
- 叠加图片功能（7种几何形状裁切 + 透明度）
- 画布比例调整（原始/1:1/4:5/3:4/9:16/16:9/4:3）
- 画布背景色自定义
- AI 主体识别（背景/前景分离）
- 手势操作（拖拽/双指缩放/旋转）
- 高质量导出（Web Share API）
- PWA 可安装
- agency-mono 设计系统（JetBrains Mono + 酸性荧光绿）
